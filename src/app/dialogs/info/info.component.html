Connect / Launching OpenGOAL

Launch OpenGOAL from the client to properly connect them together, this process can take a while (~1 min on older PCs) and is automatically done in the following steps:
(1/2) Starting OpenGOAL: A notification will be given in the client once completed.
(2/2) Connecting Teamrun to OpenGoal: Notification + Jak's color in game will go from red to normal.

If the client hasn't connected fully yet a will be displayed (for you only) next to your name in run lobbies.

OpenGOAL can be relauched at any point from the menu opened by the menu icon: in the top left of the client, a relaunch can sometimes help if the connection process seems to have failed, you can also update the OpenGOAL folder path from the menu.


Modes

Speedrun Normal team based speed run, see "Shared Items and Tasks" section for things shared between teammates.
Lockout Cells are shared with opposing teams including any unlocks that comes with them, but the cells collected by your team only count toward your own teams cell score, final boss unlocks after 73 total cells, the team with highest cell score at any point after that will have access to final boss.

Default Modifications

These are modifications applied to runs by default, they can be toggled off if desired when creating a run.
Zoomer sections between hubs requires every runner in the team to be on their zoomer before full speed for it is activated, this only needs to be done once.
Cell costs are doubled by default when sharing cells with others.
