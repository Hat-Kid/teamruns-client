Connect / Launching OpenGOAL

Launch OpenGOAL from the client to propery connect them together, this process can take a while (~1 min on older PCs) and is done in the following steps:
(1/3) Starting OpenGOAL: A notification will be given in the client once completed.
(2/3) Connecting client to OG: Notification + Jak being tripped and given red eco in game.
(3/3) Connecting Tracker to OG: Notification + Jak being tripped and given yellow eco.

If the client hasn't connected fully yet a will be displayed (for you only) next to your name in run lobbies.

OpenGOAL can be relauched at any point from the menu opened by the menu icon: in the top left of the client, you can change the path to your OpenGOAL folder from there as well.


Modes

Speedrun Normal run with items and tasks shared.
Lockout Cells are shared with opposite teams including unlocks that comes with them, final boss unlocks after 73 cells, the team with most cells at any point after that have access to final boss.

Run Settings / Defaults

Require same level: Disables cell pickup unless every teammate is in the same level.
Allow solo zoomer use: Zoomer sections between hubs requires every runner in the team to be on their zoomer before full speed for it is activated, this only needs to be done once.
Normal cell cost: Cell costs are doubled by default when sharing cells with others.
